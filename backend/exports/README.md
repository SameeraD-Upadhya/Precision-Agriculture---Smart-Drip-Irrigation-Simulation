# Exports

This folder is the **default save location** for all files exported from the
AgroFlow Pro simulation app.

## File Naming Convention

| Type | Pattern                                   |
|------|-------------------------------------------|
| JSON | `agroflow-<project>-<YYYY-MM-DD>.json`    |
| PNG  | `agroflow-<project>.png`                  |
| PDF  | `agroflow-report-<project>.pdf`           |

## How to Save Here

When the app's Export dialog opens:
- **JSON** — a save-file picker opens. Navigate to this folder and save.
- **PNG / PDF** — browser downloads to your default Downloads folder.
  Move the file here manually for organisation.

> Tip: Pin this folder to your Quick Access / Favorites in File Explorer
> so it appears instantly in the save dialog.
